# duel_master_assist

duel master  辅助挂机脚本，适用于steam版本程序, 脚本稳定性高, 几乎不会卡死
支持：超量活动挂机(等被对面打死或对面自爆)

#### 环境要求

- steam版duel_master
- 游戏窗口分辨率1280*720，高画质
- 根据requirements.txt安装必要python库

#### 使用

- 复制config.json.template到config.json, 无需修改即可使用
- 执行main.py后立即将游戏窗口前置
